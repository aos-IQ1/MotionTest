= Motion test

== 概要
ロボットのモーションを制御するサンプル。
`s.rb` に書いたような手順で行う。

== Setup

[source, bash]
----
$ bundle install --path vendor/bundle
----

== Run

[source, bash]
.run ( `4a42915ec3771ad2be3f038b9bfc3aac52688e30` 以前)
----
$ bundle exec ruby s.rb /dev/ttyUSB0
----
